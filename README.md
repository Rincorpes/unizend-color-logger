# unizend-color-logger
Easiest way to get your node.js console styled

## Instalation

  npm i --save unizend-color-logger

## Suported colors

* Black
* Red
* Green
* Yellow
* Blue
* Cyan
* White

## Usage

As easy as

  const uzl = require(unizend-color-logger)

  uzl.log('blue', 'Hello, World!')
